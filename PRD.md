# 교육용AI어시스턴트 PRD

## 1. 목표
학교 문서와 규정을 바탕으로 교내 교사가 필요한 정보를 쉽게 찾을 수 있는 AI 챗봇을 만든다.

## 2. 사용자
- 교내 교사

## 3. 핵심 기능
- PDF 문서 기반 질의응답
- 답변 근거 출처 표시
- 학교 규정/학사일정/출결 서류 안내
- API 키 입력 후 Gemini 기반 응답 생성

## 4. 현재 완료
- PRD 정리
- Streamlit 챗봇 기본 화면 구현
- PDF 문서 로딩
- FAISS 인덱스 저장 및 불러오기
- Gemini 임베딩과 답변 체인 연결
- 출처 파일명과 페이지 번호 표시
- 저장된 인덱스 빠른 재사용
- PDF 변경 감지용 manifest 기록
- 벡터 검색과 키워드 직접 검색을 합친 보강 검색
- 접이식 상태 표시 UI

## 5. 진행 중
- 답변 품질 개선
- 실제 질문 테스트
- 문서별 검색 정확도 확인

## 6. 앞으로 할 일
- 배포 환경 설정
- 예외 상황 안내 개선
- 사용자 테스트
- 학교별 문서 교체/추가 방식 정리
- 자주 실패하는 질문 목록 기반 검색 품질 평가

## 7. 대기/막힘
- Google Gemini API 키 관리 방식 확정
- 배포 주소 확정

## 8. 성공 기준
- 사용자가 학교 규정 질문을 했을 때 관련 문서 출처와 함께 답변을 받을 수 있다.
- 문서에 없는 내용은 지어내지 않고 유사 근거만 안내한다.
